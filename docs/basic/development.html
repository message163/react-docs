React开发环境搭建 ​

选择一个你喜欢的目录，然后运行以下命令：

sh
npm init vite
  • 执行完成之后会让你输入项目名称 例如 react-demo
  • 接下来会让你选择一个框架 这时候选择 react
  • 然后选择 TypeScript + SWC 如果你不会ts就选择js
